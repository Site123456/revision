Les Suites

Suites arithmétique – géométrique

Définition générale
Une suite est une fonction définie sur ℕ (ensemble des entiers naturels) qui associe à tout entier n un réel un.
Suite arithmétique
Définition :

Une suite un est arithmétique s’il existe un réel r tel que :

un+1  =  un+r

La formule explicite d'une suite arithmétique est donnée par :

un  = u0 + n · r


avec:
un est le terme de rang n,
u0 est le premier terme de la suite,
r est la raison de la suite.
n est un entier naturel.

Exemple :

u0 =  5 ,   r = 2 ⇒ un =  5 + 2n
Suite géométrique
Définition :

Une suite un est géométrique s’il existe un réel q tel que :

un+1 = un·q

La formule explicite d'une suite géométrique est donnée par :

un = u0 · qn


avec:
un est le terme de rang n,
u0 est le premier terme de la suite,
q est le quotient de la suite.
n est un entier naturel.

Exemple :

u0 =  5 ,   q = 2 ⇒ un =  5 × 2n
Suite récurrente
Définition :

Une suite est dite récurrente si chaque terme est défini en fonction de termes précédents. Exemple : un+1 = f(un).

Suite convergente
Définition :

Une suite converge si elle tend vers une valeur limite lorsque n tend vers l'infini. Si limn→∞ un = L, alors la suite est convergente et L est la limite.

Suite divergente
Définition :

Une suite est divergente si elle n'a pas de limite finie à mesure que n tend vers l'infini. Cela inclut les suites qui tendent vers +∞, -∞, ou oscillent.

Suite bornée
Définition :

Une suite est bornée si il existe un réel M tel que |un| ≤ M pour tout n.

Suite strictement croissante
Définition :

Une suite est strictement croissante si un+1 > un pour tout n.

Suite strictement décroissante
Définition :

Une suite est strictement décroissante si un+1 < un pour tout n.

Suite stationnaire
Définition :

Une suite est stationnaire (ou constante) si tous ses termes sont égaux, c'est-à-dire un = u0 pour tout n.

Suite arithmétique
Définition :

Une suite est arithmétique si chaque terme est égal au précédent plus une constante r : un+1 = un + r.

Suite géométrique
Définition :

Une suite est géométrique si chaque terme est égal au précédent multiplié par une constante q : un+1 = un × q.

Suite de Fibonacci
Définition :

Une suite définie par u0 = 0, u1 = 1, et un+2 = un+1 + un pour tout n ≥ 0.

Suite arithmético-géométrique
Définition :

Une suite qui combine les deux propriétés arithmétiques et géométriques. Par exemple : un+1 = un + r × qn, où la raison varie en fonction de n.

Suite exacte
Définition :

Une suite est exacte si elle satisfait à une relation de récurrence bien définie, souvent utilisée dans les mathématiques pures et les équations différentielles.

Suite alternée
Définition :

Une suite où les termes changent de signe à chaque étape, par exemple : un = (-1)n.

Suite Cauchy
Définition :

Une suite est Cauchy si, pour tout ε > 0, il existe un N tel que |un - um| < ε pour tout n, m ≥ N. Cela signifie que les termes deviennent arbitrairement proches entre eux.

Suite majorée et minorée
Définition :

Une suite est majorée (respectivement minorée) s'il existe un réel M (respectivement m) tel que un ≤ M (respectivement un ≥ m) pour tout n.

Suite d'uniforme
Définition :

Une suite est uniforme si elle converge uniformément vers une fonction ou une valeur.

Suite et série
Définition :

Une série est la somme infinie des termes d'une suite, souvent notée ∑n=0∞ un. Une suite convergente peut entraîner une série convergente, et inversement.
